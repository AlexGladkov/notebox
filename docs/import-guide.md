# Руководство по импорту Markdown файлов

## Обзор

Функция импорта позволяет загружать Markdown файлы в NoteBox с сохранением структуры и связей.

## Поддерживаемые форматы

- `.md` - отдельные Markdown файлы
- `.zip` - архивы с Markdown файлами и структурой папок

## Функции

### 1. Drag & Drop интерфейс

Просто перетащите файлы в область загрузки или нажмите для выбора файлов.

### 2. Превью перед импортом

Перед импортом вы увидите:
- Список всех найденных файлов
- Превью содержимого каждого файла
- Обнаруженные wiki-ссылки
- Возможность выбрать файлы для импорта

### 3. Сохранение иерархии

При импорте ZIP-архива:
- Папки автоматически создаются как заметки с иконкой 📁
- Вложенность сохраняется
- Файлы получают иконку 📄

### 4. Wiki-ссылки

Синтаксис `[[Название заметки]]` автоматически преобразуется в ссылки:
- Если заметка с таким названием существует - создается активная ссылка
- Ссылки работают в обе стороны (backlinks)

### 5. Поддержка Markdown

Корректно импортируются:
- Заголовки всех уровней (#, ##, ###)
- Списки (упорядоченные и неупорядоченные)
- Чек-боксы / задачи (- [ ] и - [x])
- Таблицы
- Блоки кода с подсветкой
- Жирный и курсивный текст
- Цитаты
- Ссылки и изображения

## Как использовать

1. Нажмите кнопку импорта (иконка загрузки) в боковой панели рядом с "Страницы"
2. Выберите файлы или перетащите их в окно
3. Просмотрите превью и выберите файлы для импорта
4. Нажмите "Импортировать"
5. Дождитесь завершения импорта

## Примеры

### Пример 1: Простой импорт

Файл `заметка.md`:
```markdown
# Моя заметка

Это текст заметки с [[ссылкой]] на другую заметку.

## Список дел

- [ ] Задача 1
- [x] Задача 2
```

### Пример 2: Импорт с иерархией

Структура ZIP-архива:
```
проекты/
  web/
    vue-app.md
  mobile/
    react-native.md
заметки/
  идеи.md
  книги.md
```

Результат: структура папок полностью сохранится в виде иерархии заметок.

## Тестовые файлы

Созданы тестовые архивы для проверки:
- `/tmp/test-notes.zip` - простой набор связанных заметок
- `/tmp/test-notes-hierarchy.zip` - заметки с иерархией папок

## Ограничения

- Максимальный размер файла: зависит от браузера (обычно ~100MB)
- Поддерживаются только текстовые Markdown файлы
- Изображения и вложения пока не импортируются (планируется в будущем)

## Troubleshooting

**Проблема:** Файлы не импортируются
- Проверьте, что файлы имеют расширение `.md`
- Убедитесь, что ZIP-архив не поврежден
- Проверьте консоль браузера на наличие ошибок

**Проблема:** Wiki-ссылки не работают
- Убедитесь, что используете синтаксис `[[название]]`
- Проверьте, что название заметки совпадает точно (включая регистр)

**Проблема:** Форматирование не сохраняется
- Убедитесь, что используете стандартный Markdown синтаксис (GFM)
- Проверьте, что блоки кода заключены в тройные обратные кавычки
