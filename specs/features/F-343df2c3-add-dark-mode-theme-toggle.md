# Спецификация: Переключатель темы (тёмный/светлый режим)

## ID задачи
F-343df2c3-add-dark-mode-theme-toggle

## Краткое описание
Добавление возможности переключения между светлой и тёмной темами с опцией "по умолчанию системная". Включает UI переключателя в настройках, сохранение предпочтений пользователя и корректную стилизацию всех экранов.

---

## Текущее состояние (анализ кодовой базы)

### Уже реализовано

#### 1. Логика переключения тем (`src/composables/useTheme.ts`)
- Три режима: `'light'`, `'dark'`, `'system'`
- Сохранение в localStorage (ключ: `notebox-theme`)
- Отслеживание системных предпочтений через `matchMedia('prefers-color-scheme: dark')`
- Применение класса `dark` к `<html>` элементу
- Функции: `setTheme()`, `cycleTheme()`, `initialize()`

#### 2. Tailwind CSS конфигурация (`tailwind.config.js`)
- `darkMode: 'class'` - активация тёмного режима через класс

#### 3. CSS переменные (`src/style.css`)
- Определены переменные для светлой и тёмной темы:
  - `--bg-primary`, `--bg-secondary`, `--bg-hover`, `--bg-active`
  - `--text-primary`, `--text-secondary`
  - `--border-color`, `--primary-color`, `--danger-color`

#### 4. UI переключателя в настройках (`src/components/settings/AppearanceSection.vue`)
- Toggle switch между светлой и тёмной темой
- Checkbox "Использовать системные настройки"
- Иконки солнца и луны
- Сохранение на сервер через `authStore.updateProfile()`

#### 5. Быстрый переключатель в сайдбаре (`src/components/Sidebar.vue`)
- Кнопка циклического переключения темы (light → dark → system → light)
- Иконки для каждого режима
- Расположение в нижней части сайдбара

#### 6. Инициализация темы (`src/App.vue`)
- Вызов `useTheme().initialize()` при загрузке приложения

#### 7. Поддержка тёмной темы в компонентах
Компоненты с полной поддержкой `dark:` классов:
- `App.vue`
- `Sidebar.vue`
- `NoteEditor.vue`
- `SettingsModal.vue`
- `AppearanceSection.vue`
- `BlockEditor.vue` (с `:deep()` стилями для ProseMirror)
- `LoginView.vue`

### 2. Предотвращение "вспышки" при загрузке (приоритет: средний)

Добавить инлайн-скрипт в `index.html` для применения темы до рендеринга:

```html
<script>
  (function() {
    const theme = localStorage.getItem('notebox-theme');
    if (theme === 'dark' || 
        (theme === 'system' && window.matchMedia('(prefers-color-scheme: dark)').matches) ||
        (!theme && window.matchMedia('(prefers-color-scheme: dark)').matches)) {
      document.documentElement.classList.add('dark');
    }
  })();
</script>
```

## Edge cases и риски

### Edge cases
- Пользователь меняет системную тему, когда приложение открыто
- localStorage недоступен (приватный режим браузера)
- Пользователь авторизуется на другом устройстве с другой темой

### Риски
- Сторонние библиотеки (TipTap/ProseMirror) могут иметь свои стили
- Эмодзи и иконки могут плохо смотреться на тёмном фоне

---

## Вне scope

- Пользовательские цветовые схемы
- Выбор акцентного цвета
- Тёмная тема для PDF экспорта
- Синхронизация темы между вкладками браузера (уже работает через localStorage)
