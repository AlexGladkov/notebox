## Детали реализации

### Конвертация TipTap в HTML

Текущая реализация в `convertTipTapToHtml()` рекурсивно обходит JSON-дерево TipTap и генерирует HTML. Функция `renderNode()` обрабатывает каждый тип узла.

### Безопасность

- `escapeHtml()` - экранирование HTML-сущностей для предотвращения XSS
- `sanitizeColor()` - валидация цветов для предотвращения CSS-инъекций
- Ссылки показываются в скобках рядом с текстом для прозрачности

### Стилизация PDF

CSS стили для PDF генерируются в `generatePdfStyles()`. Используется системный шрифт, размер 12pt, формат A4.

## Edge Cases и риски

### Edge Cases

1. **Пустая заметка** - показать placeholder "Пустая заметка"
2. **Очень длинная заметка** - может занять много времени, нужен таймаут
3. **Заметка только с базой данных** - показать placeholder
4. **Специальные символы в названии** - транслитерация и санитизация
5. **Заметка без названия** - использовать "note" как имя файла

### Риски

1. **Производительность** - html2pdf.js использует canvas, что может быть медленным для больших документов
2. **Кросс-браузерность** - html2canvas может иметь различия в рендеринге
3. **Изображения CORS** - внешние изображения могут не загрузиться из-за CORS

## Вне скоупа

- Экспорт в другие форматы (Word, Markdown, HTML)
- Серверный рендеринг PDF
- Водяные знаки и защита PDF
- Настройка шрифтов
- Оглавление для длинных документов

## Примечания

Реализация выполнена на клиентской стороне с использованием библиотеки html2pdf.js, которая комбинирует html2canvas и jsPDF. Это простое решение, не требующее серверной части, но имеющее ограничения по качеству рендеринга и производительности.
