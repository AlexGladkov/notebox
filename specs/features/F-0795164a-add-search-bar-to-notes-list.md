# Спецификация: Поисковая строка в списке заметок

**ID задачи:** F-0795164a  
**Дата создания:** 2026-04-26  
**Статус:** Анализ завершён (интервью не проведено из-за недоступности бэкенда)

## Резюме

Задача: добавить поисковую строку в верхнюю часть списка заметок для фильтрации по заголовку в реальном времени.

**ВАЖНО:** Анализ кодовой базы показал, что данная функциональность уже реализована в проекте.

## Текущее состояние (уже реализовано)

### Существующие компоненты:

1. **`src/components/SearchBar.vue`** — компонент поисковой строки
   - Поле ввода с иконкой поиска
   - Поддержка тёмной темы
   - v-model для двустороннего связывания

2. **`src/composables/useSearch.ts`** — логика поиска
   - Фильтрация заметок по заголовку (case-insensitive)
   - Сортировка результатов по дате обновления (новые первые)
   - Реактивное обновление результатов

3. **`src/views/MainView.vue`** (строки 13-40) — интеграция поиска
   - SearchBar размещён в боковой панели под профилем пользователя
   - Результаты поиска отображаются вместо дерева заметок
   - Показывается иконка и заголовок заметки
   - Сообщение "Ничего не найдено" при пустых результатах
   - Клик по результату открывает заметку

### Текущий UX:
- Поиск срабатывает при вводе (real-time)
- Поиск только по заголовку заметки
- Клиентская фильтрация (без запросов к серверу)
- При пустом поле — отображается древовидный список всех страниц
