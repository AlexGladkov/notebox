# Диагностический отчёт: Caption области нечитаемы в тёмной теме

## Краткое описание бага

Caption текст в информационном блоке демо-режима (DemoBanner) имеет критически низкий контраст в тёмной теме — светло-жёлтый текст `#fef3c7` на оранжевом фоне `#f59e0b-#d97706` практически нечитаем.

---

## Корневая причина

**Файл:** `src/components/layout/DemoBanner.vue`  
**Строки:** 91-93

```css
:global(.dark) .demo-banner-icon,
:global(.dark) .demo-banner-text {
  color: #fef3c7;
}
```

### Проблема контраста

| Элемент | Цвет текста | Цвет фона | Контраст | Статус |
|---------|-------------|-----------|----------|--------|
| DemoBanner (dark) | `#fef3c7` (светло-жёлтый) | `#f59e0b` (оранжевый) | ~1.9:1 | **FAIL** (требуется 4.5:1) |
| Кнопка "Выйти" (dark) | `#fef3c7` | `#d97706` | ~2.2:1 | **FAIL** |

**Почему это произошло:**
- В светлой теме используется `#78350f` (тёмно-коричневый) на жёлтом фоне — хороший контраст
- В тёмной теме разработчик инвертировал цвета без проверки контраста
- Предположение "светлый текст на тёмном фоне" было ошибочным — фон остался ярким оранжевым

## Находки субагентов

### Architect (Code Tracer)

**Согласие с корневой причиной:** Да

**Ключевые находки:**
1. Выявлены 6+ компонентов с hardcoded цветами `#fef3c7` в тёмной теме
2. Проблема введена при добавлении Kanban-функционала (commit e81b291, 2026-05-11)
3. Отсутствует централизованная система цветовых переменных для warning/info состояний
4. Несогласованное применение `:global(.dark)` селекторов

**Затронутые файлы с `#fef3c7`:**
- `src/components/layout/DemoBanner.vue` (строки 93, 98-99)
- `src/components/BlockEditor.vue` (строки 1201, 1316)

- `src/components/auth/DemoButton.vue` (строка 63)
- `src/components/settings/NotificationsSection.vue` (строка 243)

### Stack Expert (Framework Analysis)

**Согласие с корневой причиной:** Да

**Ключевые находки:**
1. Технологический стек: Vue 3 + Tailwind CSS v3.4.0 + Vite 5.0
2. Тёмная тема реализована через `darkMode: 'class'` в Tailwind
3. Дополнительная проблема: паттерн `dark:text-gray-400` / `dark:text-gray-500` используется в 25+ компонентах
4. Tailwind цвета `gray-400` (#9ca3af) и `gray-500` (#6b7280) не проходят WCAG AA на тёмных фонах

## План исправления

### Шаг 1: Исправить DemoBanner.vue (критический)

**Файл:** `src/components/layout/DemoBanner.vue`

**Действие:** Изменить цвет текста в тёмной теме с `#fef3c7` на `#78350f` (тёмно-коричневый)

```css
/* Было: */
:global(.dark) .demo-banner-icon,
:global(.dark) .demo-banner-text {
  color: #fef3c7;
}

/* Должно быть: */
:global(.dark) .demo-banner-icon,
:global(.dark) .demo-banner-text {
  color: #78350f;
}
```

**Контраст после исправления:** `#78350f` на `#f59e0b` = ~4.8:1 (проходит WCAG AA)

### Шаг 2: Исправить кнопку "Выйти" в DemoBanner

**Строки 96-99:**
```css
/* Было: */
:global(.dark) .demo-banner-logout {
  background: rgba(254, 243, 199, 0.1);
  border-color: #fef3c7;
  color: #fef3c7;
}

/* Должно быть: */
:global(.dark) .demo-banner-logout {
  background: rgba(120, 53, 15, 0.15);
  border-color: #78350f;
  color: #78350f;
}
```

### Шаг 3: Исправить CreateViewModal.vue warning-box

**Файл:** `src/components/BlockEditor/CreateViewModal.vue`

**Строки 350-355:**
```css
/* Было: */
.dark .warning-box {
  background: #78350f;
  color: #fef3c7;
}

/* Должно быть: */
.dark .warning-box {
  background: #fef3c7;
  color: #78350f;
}
```

### Шаг 4: Исправить BlockEditor.vue (highlight/mark стили)

**Файл:** `src/components/BlockEditor.vue`

**Строки 1200-1201, 1314-1316:** Изменить `color: #fef3c7` на `color: #78350f` или использовать более тёмный фон с светлым текстом.

### Шаг 5 (опционально): Систематическое исправление Tailwind-классов

Заменить паттерн `dark:text-gray-400` на `dark:text-gray-200` или `dark:text-gray-300` в затронутых компонентах для улучшения контраста.

## Заключение

Баг подтверждён визуально. Корневая причина — использование светло-жёлтого цвета `#fef3c7` для текста на оранжевом фоне в тёмной теме компонента DemoBanner. Исправление требует изменения цвета текста на тёмно-коричневый `#78350f` для обеспечения контраста по стандарту WCAG AA (минимум 4.5:1).
